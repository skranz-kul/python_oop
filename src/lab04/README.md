# ЛР-4 — интерфейсы и абстрактные классы (abc)

## 1. цель работы

в лабораторной работе изучены абстрактные базовые классы и интерфейсы как контракты поведения между компонентами системы

## 2. описание интерфейсов

в `src/lab04/interfaces.py` созданы 2 интерфейса:

- `Printable`
  - обязательный метод: `to_display_string() -> str`
- `Comparable`
  - обязательный метод: `sort_key() -> tuple[object, ...]`

оба интерфейса оформлены через `ABC` и `@abstractmethod`

## 3. реализация в классах

в `src/lab04/models.py` интерфейсы реализуют:

- `PrintedBook`
- `Ebook`
- `AudioBook`

отличие поведения:

- `to_display_string()` формирует разный вывод для каждого типа книги
- `sort_key()` возвращает разные ключи сортировки в зависимости от типа объекта

дополнительно реализован `Book` с базовой интерфейсной логикой
